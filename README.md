# BI Briefing

BI Briefing er en Progressive Web App til CFO'er og ledere, der samler centrale nøgletal, viser afvigelser og genererer en kort dansk AI-ledelsesbriefing.

## Kom i gang

```bash
npm install
npm run dev
```

## Miljøvariabler

Opret `.env.local` med:

```bash
ANTHROPIC_API_KEY=sk-ant-your-key-here
ANTHROPIC_MODEL=claude-sonnet-4-6
```

## Deploy

Push til `main`, så deployer Vercel automatisk fra GitHub.
